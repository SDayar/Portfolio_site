Projets

Cette rubrique présente les principaux projets effectués durant ma première année de formation

Nolark : e-commerce de casques moto

Nolark est un projet de développement d'un site web Full-Stack. Ce projet est repartie en trois grandes parties qui, elles-mêmes, en plus de 10 TP. Ce projet permet d'avoir un aperçu sur les notions de la base sur la programmation web en se servant des langages suivant : HTML&CSS, Javascript et PHP.

  • - Site web Full stack : Front - end & Back-end
  • - Site web responsive avec CSS et dynamique avec Java script
  • - Manipulation d'une base de données
  • - Langages : HTML/CSS/Javascript/PHP/SQL
  • - Outils : Netbeans et phpMyAdmin

Projet professionnel encadré : PPE

Il s'agit d'un projet réalisé en groupe dont j'ai contribué en tant que "chef de projet". Ce projet consiste à réaliser un système d'information d'une filiale qui vient de s'installer à Paris. Ce projet est divisée en trois TP, eux mêmes divisés en diverses tâches :

  • - Enumération des matériaux de gestions nécessaires à l'agence : postes client
  • - Installation de l'iso Windows 7 dans la machine virtuelle Oracle VM VirtualBox
  • - Création d'un site web Front-end et dynamique avec Javascript
  • - Création et interconnexions de réseaux qui convergent au siège
  • - Base de données : Schema conceptuel, logique et relationnel des données
  • - Framework : Font-awesome
  • - Outils : Netbeans,Cisco-Packet Tracer, WinDesign et phpMyAdmin

Jeu du nombre caché

Ce projet est réalisé afin de mettre en pratique les connaissances acquises en Python. Le but de ce projet est de réaliser un programme qui permet à l'utilisateur ou à la machine de proposer un nombre puis de laisser à l'autre (soit la machine ou à l'utilisateur ) de deviner ce nombre là. En réalisant ce projet, j'ai étendu ma perception sur les structures des algorithmes. Ces derniers reposaient sur les outils suivants:

  • - Bibliothèque : Random
  • - Editeur : Thonny

Création de réseaux avec Cisco Packet Tracer

Ce projet constitue ma première initiation au domaine du réseau. Il est divisé en plusieurs TP, eux mêmes divisés en plusieurs tâches complémentaires.

  • - Réseau LAN ( Per to Per)
  • - Réseau LAN / PAN
  • - Réseau plus élargi ( MAN et WAN)

Création d'un logiciel : programmation évenementielle

Ce projet est le dernier effectué avec le langage C# durant ma première année. Il consiste à effectuer un logiciel simple qui permet de sélectionner puis afficher le contenu d'un dossier depuis le disque de stockage de la machine.

  • - Langage : C#
  • - Framework : .Net

Manipulation de l'outil Openssl

Ce projet rassemble plusieurs TP qui m'ont permis de mettre en pratique toutes les théories de la cryptographie et la cryptanalyse .

  • - Algorithmes : des3, aes-256-cbc, rsa, dgst et pkeyutl
  • - Chiffrement symétrique et asymétrique
  • - Signature, vérification et Hachage
